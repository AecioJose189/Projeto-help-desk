# [Projeto faculdade] Projeto do 5° período da faculdade - Help Desk
> **Cadeira:** Banco de dados 
> **Universidade:** UFRPE - Universidade Federal Rural de Pernambuco  
> **Alunos:** Aécio José e Gabriel Germano

## Qual a ideia do projeto?
A ideia deste projeto é criar um banco de dados bem povoado de um sistema help desk para a cadeira de Banco de dados.

## Como Rodar?
### Abaixo está o passo a passo de como instalar as bibliotecas e rodar o banco

- Baixar o MySQL
- Instalar o SQL
- Fazer as configurações
- Rodar as tabelas e depois as funções e triggers
